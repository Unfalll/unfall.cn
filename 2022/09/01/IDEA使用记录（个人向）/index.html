IDEA常用快捷键

  1. 删除当前行，自己配置CTRL + d

  2. 复制当前行，配置 CTRL + alt + 向下光标

  3. ctrl + alt + T 常用快捷语句（if，for。。。）

  4. 补全代码 alt + /

  5. 添加注释和取消注释 CTRL + / 【第一次是添加注释，第二次是取消注释】

  6. Diagrams下,选中类,ctrl + alt + B,创建子类

  7. 查看基础类 ctrl + h

  8. ctrl + R 替换内容
