IDEA常用快捷键

  1. 删除当前行，自己配置CTRL + d
  2. 复制当前行，配置 CTRL + alt + 向下光标
  3. 补全代码 alt + /
  4. 添加注释和取消注释 CTRL + / 【第一次是添加注释，第二次是取消注释】

模板/自定义模板

file -> settings -> editor -> Live templates -> 查看有那些模板快捷键/可以自己增加模板
